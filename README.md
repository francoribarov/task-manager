# Task Manager

Task Manager is a [Node](https://nodejs.org/es) and [Express](https://expressjs.com) webpage that uses [MongoDB](https://www.mongodb.com). 
It allows the user to create, manage and complete tasks. 

## Briefly description of the project.

This project is made with educational purposes.

The project allows the user to manipulate a list of tasks that have persistance in Mongo database.

## How to run the project

1) Clone the repo.
2) Use npm install, to install all the dependencies of the project.
3) Create a .env file, in which specify the link of Mongo database in a variable called MONGO_URI.
4) Run npm start to launch the project 
5) Open a browser and go to localhost:3000
